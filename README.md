# Magma | Real Estate Agility

Magma is a high-end, Web3-powered real estate platform that transforms physical buildings into liquid digital assets. This project is a premium, high-performance landing page featuring advanced scrollytelling, canvas animations, and a modern design system.

## âœ¨ Key Features

- **Advanced Scrollytelling**: High-performance image sequence animations controlled by user scroll using GSAP and ScrollTrigger.
- **Premium Preloader**: A cinematic loading screen with a live percentage counter and smooth reveal transitions.
- **Custom Dual-Layer Cursor**: A precise dot and trailing ring cursor with intelligent hover interactions.
- **Section Reveal Effects**: Smooth scroll-triggered entrance animations for all sections using Intersection Observer.
- **Global Portfolio**: Interactive property cards with glassmorphism effects, status labels, and detailed building stats.
- **Modern UI/UX**: Dark mode aesthetic, premium typography, and responsive layouts.
- **Smooth Scrolling**: Implemented using Locomotive Scroll for a fluid, natural feel.
